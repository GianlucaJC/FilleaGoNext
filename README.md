# FilleaGO Next

Una riscrittura moderna dell'applicazione legacy, costruita con Laravel e Vue.js per gestire e le informazioni FilleaGO.

## Informazioni sul Progetto

Questo progetto è un refactoring completo di una vecchia applicazione originariamente costruita con CodeIgniter. L'obiettivo è fornire una piattaforma moderna, responsive e manutenibile.
